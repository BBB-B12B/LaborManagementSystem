# CFP Draft: Walkthrough Artifact Path Error

## Trigger
Writing or editing the `walkthrough.md` artifact using path syntax that fails to render correctly or causes file resolution errors on Windows.

## Root Cause
The path of the walkthrough artifact was not generated or formatted as a valid absolute URI (`file:///...` with forward slashes) or failed to follow Windows absolute path conventions, causing a path resolution error.

## Correct Resolution
When creating or editing `walkthrough.md`, ensure the target path is strictly formatted as a standard Windows absolute path with forward slashes, and references to it utilize the absolute `file:///` URI scheme.
